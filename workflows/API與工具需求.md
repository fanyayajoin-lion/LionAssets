# API 與工具需求

本檔只記錄需要哪些工具與環境變數，不保存實際 API Key。

| 工具 | 用途 | 建議環境變數 | 狀態 |
| --- | --- | --- | --- |
| DeepSeek | 蒸餾、摘要、低成本批次處理 | `DEEPSEEK_API_KEY` | 待設定 |
| GitHub | 讀取 repo、追蹤 AI 專案 | `GITHUB_TOKEN` | 待設定 |
| APIFY | 抓取 X、Reddit、網頁資料 | `APIFY_API_TOKEN` | 待設定 |
| Firecrawl | 網頁搜尋、爬取、整理 markdown | `FIRECRAWL_API_KEY` | 待設定 |
| Telegram Bot | 手機剪藏入口 | `TELEGRAM_BOT_TOKEN` | 待設定 |
| Obsidian Clipper | 電腦剪藏 | 本機設定 | 待設定 |
| Xiaohongshu Importer | 小紅書案例匯入 | 本機設定 | 待設定 |

新增工具時，先確認是否真的會重複使用；一次性工具不要寫進核心流程。
