# 每週內容工廠 Workflow

## 使用時機

每週從知識庫整理高價值觀點，產出 Threads、Facebook、IG 輪播草稿。

## 核心原則

- `outputs/` 不是知識終點，輸出後要回流 wiki。
- 先產草稿，不自動發文。
- 內容要符合業萱文風：真誠分享、有實業背景、不是站在台上教人。

## 輸出來源

搜尋順序：

```text
hot.md -> wiki/synthesis -> wiki/concepts -> wiki/notes -> inbox/processed
```

## 輸出位置

- Threads：`outputs/threads/`
- Facebook：`outputs/facebook/`
- IG 輪播：`outputs/ig-carousel/`

## 每週基本產出

- 5 則 Threads。
- 2 篇 Facebook 長文。
- 1 組 IG 輪播大綱。

## 回流規則

每個 output 都要標記：

- 使用來源 note / concept / synthesis。
- 產出主題。
- 是否值得延伸成 skill / workflow。
- 是否需要更新 `hot.md`。

Discord `#weekly-output` 顯示本週草稿摘要。
