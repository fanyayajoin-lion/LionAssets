# 剪藏整理 Workflow

## 目的

整理電腦網址、圖片、Obsidian Clipper、Telegram、Xiaohongshu Importer 等剪藏資料。

## 步驟

1. 原始資料先進 `raw/` 或 `assets/`，不可修改原檔。
2. 網址、文章、YT、圖片、語音需補 metadata。
3. 摘要後進 `wiki/notes/`。
4. 若是品牌案例，改走 `workflows/品牌案例補全歸檔.md`。
5. 評 Lion Score。
6. 更新 `index.md`、`hot.md`、`log.md`。

## Metadata

```yaml
type:
source:
captured_at:
processed_at:
status: 待整理 / 已整理 / 需查證
```
