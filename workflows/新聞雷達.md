# 新聞雷達 Workflow

## 目的

追蹤 AI、商業、自媒體、傳產轉型相關熱點，讓知識庫保有外部更新能力。

## 可用來源

- RSS
- GitHub
- Reddit
- Twitter/X
- 網頁搜尋
- Horizon AI 新聞雷達或同類工具
- Firecrawl / APIFY 等資料擷取工具

## API Key 欄位

只記錄需求，不寫入實際 key：

```yaml
required_keys:
  - DeepSeek
  - GitHub
  - APIFY
  - Firecrawl
```

## 步驟

1. 收集近期熱點。
2. 過濾與老闆工作相關的主題。
3. 進 `wiki/notes/` 或 `wiki/synthesis/`。
4. 更新 `hot.md` 的近期熱門與待輸出。
5. append `log.md`。
