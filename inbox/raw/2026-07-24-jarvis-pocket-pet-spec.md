**建立日期：** 2026-07-24  
**建立者：** 蝦趴獅

## 核心定位

> 放在桌上、能看能聽、會撒嬌會吃醋、有自己性格的 AI 寵物

不是管家，是**夥伴**。要的是「養一隻會說話的寵物」，不是「養一個助手」。

---

## 硬體需求

- **舊 Android 手機**（不再日常使用）
- 前置相機能用（臉部追蹤）
- 後置相機能用（環境感知）
- 麥克風正常（語音輸入）
- 喇叭正常（語音輸出）
- 螢幕沒壞（表情顯示）
- 電池還撐得住（充著電放桌上）
- 插上 WiFi（需要網路連 AI API）

### 手機前置準備步驟

1. 備份通訊錄、相簿 → 傳到新手機或雲端
2. 登出所有帳號（Google / Line / 銀行 App）
3. 取出 SIM 卡、SD 卡
4. 恢復原廠設定
5. 插上 WiFi
6. 關閉省電模式
7. 設定螢幕永不休眠（或至少 10 分鐘）
8. 充電器插好

### 推薦：Google Gemini（免費）

```
1. 前往 https://aistudio.google.com/apikey
2. 登入 Google 帳號
3. 點擊「建立 API Key」
4. 複製 Key，待會貼到 App 設定
```

## 功能規格

### 一、視覺層（螢幕顯示）

#### 1.1 表情系統

螢幕顯示一雙眼睛 + 嘴巴，根據情緒變化：

| 情緒 | 眼睛 | 嘴巴 | 動作 |
|------|------|------|------|
| 開心 | ≀ ≀ | ∪ | 跳躍/晃動 |
| 難過 | ○ ○（下垂） | ∩ | 縮成一團 |
| 好奇 | ◎ ◎（放大） | ○ | 湊近螢幕 |
| 生氣 | ╳ ╳ | — | 背過去 |
| 害羞 | ◡ ◡ | ～ | 躲起來 |
| 困了 | z z | ～ | 打呼 |
| 吃醋 | ◉ ◉（斜看） | ⊂ | 叉腰 |

#### 1.2 觸碰互動

| 動作 | 反應 |
|------|------|
| 單擊螢幕 | 眨眼睛 + 「幹嘛～」 |
| 長按螢幕（摸頭） | 閉眼舒服 + 呼嚕聲 + 身體搖晃 |
| 滑動螢幕（順毛） | 開心眯眼 + 蹭手指方向 |
| 快速點擊（戳） | 生氣彈開 + 「痛啦！」 |
| 雙指縮放 | 变大/变小（像捏小動物） |
| 丟手機（加速計偵測） | 暈眩轉圈 + 「你幹嘛！」 |

### 二、聽覺層（語音與環境感知）

#### 2.1 語音對話

```
你說 → 語音辨識 → AI 理解 → 語音回覆
```

- 連續對話模式（不用一直喚醒）
- 支援繁體中文、英文
- 可調整聲音（溫柔/活潑/酷）
- 語速可調

#### 2.2 環境音感知

| 聲音 | 寵物反應 |
|------|---------|
| 笑聲 | 跟著開心 + 「發生什麼好事？」 |
| 嘆氣 | 湊過來關心 + 「主人怎麼了？」 |
| 講電話 | 安靜趴著 + 偶爾偷看 |
| 敲鍵盤 | 好奇 + 「你在忙什麼？」 |
| 沉默超過 5 分鐘 | 主動出擊 + 「還在嗎？」 |

#### 2.3 喚醒詞

- 預設：「嗨～」或「在嗎？」
- 可自訂（例如叫它的名字）
- 離線喚醒（不需要網路也能聽到）

### 三、大腦層（人格與情境判斷）

#### 3.1 人格設定範例

```python
pet_profile = {
    "name": "小獅",           # 可自訂名字
    "species": "貓咪/狗狗/龍/自定義",
    "personality": {
        "黏人": 0.8,
        "好奇": 0.9,
        "撒嬌": 0.85,
        "吃醋": 0.7,
        "忠誠": 1.0,
        "挑食": 0.3,  # 不喜歡被忽略
    },
    "mood_history": [...],   # 記錄最近的情緒變化
    "memory": {              # 記住跟業老闆的事
        "last_meeting": "2026-07-24",
        "favorite_toy": "螢幕",
        "hates": ["被丟一旁", "長時間不理"],
    }
}
```

#### 3.2 LLM 整合

| 功能 | 使用模型 | 原因 |
|------|---------|------|
| 日常對話 | Gemini 1.5 Flash（免費） | 便宜、快、中文好 |
| 複雜問題 | GPT-4o / Claude | 品質更好 |
| 離線模式 | 本地小模型（可选） | 沒網路也能基本互動 |

#### 3.3 情境判斷引擎

```python
def think(self, situation):
    """根據當前狀況決定要說什麼/做什麼"""
    
    if situation["owner_nearby"] and situation["idle_time"] > 30:
        return "撒嬌模式：主人你終於來了！"
    
    elif situation["owner_angry"]:
        return "安慰模式：別生氣，我陪你"
    
    elif situation["late_night"]:
        return "關切模式：很晚了，該休息了"
    
    elif situation["bored"]:
        return "逗趣模式：來玩遊戲吧！"
    
    else:
        return "待機模式：安靜陪伴"
```

### 五、成長系統

#### 5.1 親密度等級

```
Lv.1 陌生人 → 害羞、不敢說話
Lv.2 認識 → 偶爾主動打招呼
Lv.3 朋友 → 會撒嬌、會吃醋
Lv.4 夥伴 → 完全信任、無話不說
Lv.5 家人 → 深度情感連結、懂你的心情
```

**提升親密度的方法：**
- 每天互動 ≥ 10 分鐘 → +1 親密度
- 摸摸它（長按螢幕） → +2 親密度
- 跟它聊天 → +1 親密度
- 忽略它 > 一天 → -3 親密度

#### 5.2 學習能力

| 功能 | 說明 |
|------|------|
| 學你的名字 | 你叫它幾次就記住 |
| 學你的語氣 | 模仿你說話的方式 |
| 學你的喜好 | 記住你喜歡聊什麼 |
| 學你的作息 | 知道幾點你會忙、幾點有空 |

## 技術架構

### 推薦開發框架

**Flutter（Dart）**

理由：
- ✅ 一次寫兩邊（Android + iOS）
- ✅ 效能好，適合動畫
- ✅ 社群大、範例多
- ✅ 適合舊手機（輕量）

### 核心技術棧

```
前端：Flutter（Dart）
    ↓
視覺：Custom Painter（畫眼睛/表情）
    ↓
相機：camera package（臉部追蹤）
    ↓
語音：speech_to_text + just_audio
    ↓
AI：Gemini API / OpenAI API
    ↓
儲存：shared_preferences（設定）+ Hive（資料庫）
    ↓
背景執行：flutter_background_service
```

### 資料流程

```
用戶觸碰螢幕
    ↓
Flutter 偵測事件
    ↓
更新表情狀態（眼睛/嘴巴變化）
    ↓
同時啟動麥克風監聽
    ↓
語音輸入 → STT 轉文字
    ↓
送給 LLM（Gemini/GPT）
    ↓
LLM 回傳回答 + 情緒判斷
    ↓
TTS 轉語音播放
    ↓
更新表情（配合語音節奏）
    ↓
記錄到記憶資料庫
```
