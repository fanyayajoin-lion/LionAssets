# 創業檢查清單實戰 — 香料大叔 45/100 與金飾 20/100 風險對比分析

## 來源

2026-07-11 下午，業萱要求用新建立的「一人公司創業檢查清單」評估兩個案子：香料大叔（堉華生技食品）和金飾業。

## 香料大叔評分：45/100

### 通過項（✅）
- 目標客群明確（食品業、傳產老闆）
- 解決真實痛點（傳統產業數位轉型困難）
- 價值主張清楚（傳產翻譯機）
- 收入來源明確（顧問費 + 系統授權）
- 成本結構可控（一人公司）
- 使用者體驗符合（傳產老闆愛簡單）
- 定價策略合理
- 控制燒錢速度
- 利潤優先於規模
- 公司組織登記
- 必要執照齊全

### 部分通過（⚠️）
- 市場規模待確認（香料/生技細分市場夠不夠大？）
- 可複製性待加強（每個香料廠的需求可能不同）
- MVP 狀態不明（手動流程跑通了嗎？）
- 技術架構待確認
- 交付物定義模糊
- 現金流預測未知
- 智慧財產權待確認
- 品牌定位關係不明（「傳產翻譯機」vs「香料大叔」）
- 推廣管道待確認
- 客戶獲取流程未建立

### 未通過（❌）
- **合作夥伴生態 = 0** — 關鍵合作對象不明、互利模式不清楚、沒有測試過的合作關係
- **持續成長 = 0** — 沒有可重複模組、沒有定期檢視機制、沒有知識庫

### 高風險
1. 合作夥伴生態完全空白（沒有產業資源無法擴張）
2. 持續成長機制缺失（做一個案子就斷一個，無法累積）

### 建議行動（優先順序）
1. 定義第一個 MVP 交付物 — 什麼東西能在 2 週內做出來展示？
2. 找 1-2 個合作對象小規模試
3. 準備標準顧問合約模板
4. 選一個推廣管道集中打
5. 強制留下 SOP + 案例 + 工具包

## 金飾評分：20/100

### 幾乎全數未通過

| 項目 | 香料大叔 | 金飾 |
|------|----------|------|
| 產業了解 | 有基礎 | 完全空白 |
| 痛點確認 | 有假設 | 未驗證 |
| 人脈資源 | 有接觸 | 未知 |
| MVP 方向 | 可定義 | 完全空白 |
| 合作夥伴 | 空白 | 空白 |
| 行銷策略 | 模糊 | 空白 |

### 高風險
1. 完全不了解金飾產業（痛點、預算、法規、人脈）
2. 沒有任何合作夥伴資源
3. 沒有 MVP，不知道能不能做出東西
4. 金飾業高度依賴人脈和信任，進去門檻極高

### 黑獅建議：**先放下金飾**

正確路徑：
```
香料大叔（食品業）→ 累積案例 → 建立 SOP → 拓展到金飾
```

而不是：
```
金飾（完全陌生）→ 盲目探索 → 高風險失敗
```

## 洞察

1. **檢查清單的價值在於暴露盲點** — 香料大叔自認為有方向，但清單顯示執行細節嚴重不足
2. **跨產業擴張的風險被嚴重低估** — 金飾案子的 20 分提醒我們：不了解產業就不要碰
3. **「合作夥伴生態」是傳產轉型最容易被忽略的一環** — 傳統產業靠人脈，不是靠廣告
4. **持續成長機制是多人公司容易忽略的** — 一人公司更要注意，不做知識累積就會一直從零開始
