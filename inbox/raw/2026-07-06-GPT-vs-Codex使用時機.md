# 2026-07-06 GPT vs Codex 使用時機分享

## 背景
有人問 GPT 和 Codex 什麼時候用，分享自己的實際案例

## 案例：Hermes Agent 技術手冊
想做一份關於「Hermes Agent」的完整研究
- 目標：技術小白和使用場景都能理解
- 給人和 Agent 都能自己了解、更好執行
- 痛點：官方手冊和 GitHub 大神都是工程技術背景，看不懂

## GPT 的表現
- 前面：完美給出完整計劃 ✅
- 執行時問題：
  1. 不停建立一小部分就要停下來確認
  2. 每次停下來都會自己延伸 😂
  3. 甚至應該把檔案建完同步到 GitHub repo，都沒繼續
- 結論：受不了，整理進度後丟給 Codex

## 使用心得
**GPT** = 適合頭腦風暴、確認想法
**Codex** = 適合執行層面，手腳快
但 Codex 有額度限制
