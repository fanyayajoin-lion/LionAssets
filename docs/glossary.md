# 術語字典 — 看不懂的字看這裡

每次看到新詞，回來查這頁。

| 術語 | 白話解釋 | 比喻 |
|------|---------|------|
| **Hermes** | 你的 AI 管家 | 餐廳總經理 |
| **Provider** | AI 模型（大腦） | 廚師 |
| **Agent** | Hermes 本身（管家） | 管理員 |
| **Gateway** | 聯絡方式（Telegram、網頁…） | 掛號窗口 |
| **Skill** | 你的工作方法包 | 秘訣手冊 |
| **Memory** | 記住你的偏好 | 秘書的筆記本 |
| **Profile** | 不同的分身 | 不同專長的員工 |
| **API Key** | 進入 AI 世界的鑰匙 | 保險箱密碼 |
| **CLI** | 用打字指令操作的方式 | 用鍵盤跟電腦說話 |
| **VPS** | 24 小時不關機的雲端電腦 | 租店面 |
| **Backup** | 把資料複製到別的地方 | 買保險 |
| **Troubleshoot** | 問題發生時怎麼修 | 手機當機怎麼重啟 |
