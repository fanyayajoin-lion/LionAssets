# Human Manual｜給人看的 Hermes 手冊

這裡是給技術小白看的 Hermes Agent 手冊。

## 閱讀方式

如果你完全不懂工程，照這個順序看：

1. **hermes-overview.md**：先看 Hermes 是什麼
2. **who-is-this-for.md**：確認你是不是適合用
3. **glossary.md**：術語字典（遇到不懂的詞回來查）
4. **quick-start.md**：30 分鐘建好你的 Hermes
5. **choosing-your-brain.md**：選最適合你的 AI 模型
6. **first-conversation.md**：跟 Hermes 說第一句話

之後再看各進階章節。

## 每章固定格式

所有給人看的章節都會盡量包含：

```
一句話理解
這是什麼
為什麼需要
生活比喻
實際使用場景
操作步驟
常見錯誤
壞掉會怎樣
技術補充
```
