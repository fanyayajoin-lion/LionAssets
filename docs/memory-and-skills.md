# 記憶與技能 — Hermes 怎麼變懂你

## 一句話理解
Hermes 會**記住你**（記憶），也會**學會你的工作方法**（技能）。這兩樣東西讓他越用越懂你。

## 這是什麼

### 記憶（Memory）= 你的個人檔案
Hermes 會記住：
- 你的名字、職業、喜好
- 你喜歡繁體中文、結論先行
- 你上週說了什麼、做了什麼

下次你再跟他說話，他會說：「嗨老闆，上次你說要整理客戶資料，要繼續嗎？」

### 技能（Skill）= 你的工作方法
技能是把你的**工作本事**變成 AI 也能做的事。

比如：
- 你回覆客戶的標準流程 → 變成 Skill
- 你做阿米巴報告的格式 → 變成 Skill
- 你解命理的邏輯 → 變成 Skill

有了 Skill，Hermes 就不只是「會聊天的助理」，而是「懂你工作方法的夥伴」。

## 為什麼需要
沒有記憶的 AI = 每次見面都是陌生人。
沒有技能的 AI = 只能聊天，不能做事。

**記憶 + 技能 = 你的 AI 分身**

## 生活比喻
想像你請了一個**新秘書**：

### 記憶 = 認識你的過程
- 第一天：「你好，我叫 XX，我是顧問，我喜歡…」
- 一個月後：她記得你的喜好、習慣、客戶名字
- 半年後：你還沒開口，她就幫你做好準備

### 技能 = 學會你的工作
- 第一週：她只會回 Email
- 一個月後：她知道怎麼做報告、怎麼回覆客戶
- 三個月後：她做得跟你一樣好，甚至更好

**記憶讓她懂你，技能讓她像你。**

## 實際使用場景

### 記憶：讓 Hermes 記住你

第一次見面時告訴他：
```
你好，我是樊業萱，是一個傳產數位轉型顧問。
我喜歡結論先行，回覆不要超過 3 句話。
我的客戶主要是 40-60 歲的傳統產業老闆。
```

以後他每次都記得。

### 技能：把你的工作變成 Skill

假設你最擅長「回覆客戶諮詢」，你可以這樣教：

```
我想教你怎麼回覆客戶諮詢：

1. 先問客戶是做哪一行的
2. 再問他們目前遇到的困難
3. 根據困難推薦我們的服務
4. 回覆語氣要尊重、專業、不推銷

這是我們公司回覆客戶的標準流程，請記住。
```

Hermes 學會後，以後客戶來信，他會用你的方式回覆。

## 常見錯誤

### 錯誤 1：以為記憶是永久的
記憶存在你的電腦或雲端。如果重灌系統，記憶會消失。
**解法**：定期備份 `~/.hermes/` 目錄。

### 錯誤 2：以為技能是自動生成的
技能需要你**主動教**。Hermes 不會自己猜你的工作方法。
**解法**：把你的工作流程寫下來，一個步驟一個步驟教給他。

### 錯誤 3：以為一個 Skill 只能做一件事
一個 Skill 可以包含多個步驟和流程。
**解法**：把相關的工作打包成一個 Skill，比如「客戶諮詢 Skill」包含回覆流程、推薦邏輯、跟進提醒。

## 壞掉會怎樣
- **記憶丟了**：Hermes 不記得你了，但還是能用。重新介紹就好。
- **技能失效**：Hermes 用舊方式做事。重新教一次就好。
- **兩者都不會影響你原本的 AI 模型**（ChatGPT、Claude 還是正常的）。

## 技術補充
- 記憶位置：`~/.hermes/memories/`
- 技能位置：`~/.hermes/skills/`
- 查看記憶：`hermes memory list`
- 查看技能：`hermes skills list`
- 清除記憶：`hermes memory clear`
- 清除技能：`hermes skills uninstall <name>`

## 下一步：把你的本事變成 Skill

Hermes 的記憶和技能只是基礎。真正強大的是——**把你公司的核心能力，變成可複製的 Skill**。

比如：
- 你的阿米巴經營顧問能力 → 變成 Skill
- 你的數位轉型診斷流程 → 變成 Skill
- 你的客戶回覆標準 → 變成 Skill

這樣你就可以：
1. 讓 AI 像你的員工一樣做事
2. 讓 AI 像你的顧問一樣思考
3. 讓 AI 像你的品牌一樣回覆客戶

**這就是我們後續課程「AI 人才複製」在教的事——**
**不只是學會用 Hermes，而是學會把你的公司本事變成 AI 的技能。**
