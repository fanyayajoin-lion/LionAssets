# 語音模式設定

## 一句話理解
讓 Hermes **聽你說話、跟你說話**，不用打字。

## 這是什麼
Hermes 的語音功能有兩部分：
1. **聽你說話**（STT）— 你把麥克風靠近手機，他說什麼他聽得到
2. **跟你說話**（TTS）— 他用聲音回覆你，不是文字

## 為什麼需要
- 開車時不方便打字
- 眼睛忙著看別的東西
- 單純覺得用說的比較快

## 生活比喻
就像**語音助理**（Siri、Google 助理）：
- 你說話 → 他聽懂 → 他做事
- 他回覆 → 你用听的 → 更方便

## 操作步驟

### 步驟 1：開啟語音功能

在 Hermes 對話中輸入：
```
/voice on
```

這樣他就會聽你說話、跟你說話。

### 步驟 2：測試

對著手機說：
```
你好，請自我介紹
```

如果 Hermes 用聲音回覆你，恭喜！語音功能正常。

### 步驟 3：關閉語音

如果不想用語音了：
```
/voice off
```

## 常見錯誤

### 錯誤 1：麥克風沒聲音
**解法**：檢查手機或電腦的麥克風設定，確認有開啟權限。

### 錯誤 2：回覆聽不到
**解法**：檢查手機音量，確認不是靜音模式。

### 錯誤 3：聽不懂你說什麼
**解法**：
- 說慢一點、清楚一點
- 遠離噪音來源
- 或改用文字輸入

## 壞掉會怎樣
語音功能壞了**不影響文字對話**。Hermes 還是能用，只是不能用說的而已。

## 技術補充
- 語音模式：`/voice on` / `/voice off`
- 支援平台：Telegram、Discord、Slack
- 需要網路連線才能使用語音功能
