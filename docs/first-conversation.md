# 跟 Hermes 說第一句話

## 一句話理解
學會用正確的姿勢跟你的 AI 助理說話，他會更懂你、更好用。

## 這是什麼
Hermes 不像一般聊天機器人，他會**記住**你跟他說過的話。所以第一次見面很重要，你要讓他知道：
- 你是誰
- 你想要他做什麼
- 你喜歡什麼樣的回覆方式

## 為什麼需要
想像你請了一個新秘書：
- 你第一天就要告訴他你的喜好
- 你喜歡繁體中文還是簡體中文？
- 你喜歡他回覆長還是短？
- 你最常處理什麼類型的工作？

說得越清楚，他以後就越懂你。

## 第一句話怎麼說

### 步驟 1：自我介紹

打開 Hermes 後，輸入：

```
你好，我是樊業萱，是一個傳產數位轉型顧問。我主要幫 40-60 歲的二代接班人做 AI 落地諮詢。
```

這樣 Hermes 就會記住你的名字和職業。

### 步驟 2：設定偏好

```
請記住：
1. 我用繁體中文回覆
2. 我喜歡結論先行，不要廢話
3. 我是一個技術小白，解釋時請用白話
4. 我的工作時區是 Asia/Taipei（UTC+8）
```

### 步驟 3：說明你的需求

```
我希望你能幫我：
1. 整理客戶資料
2. 寫行銷文案
3. 追蹤專案進度
4. 每天整理一份工作報告
```

## 日常對話技巧

### 技巧 1：具體說明
**不好**：「幫我寫個東西」
**好**：「幫我寫一篇 Facebook 貼文，主題是『AI 如何幫助傳統產業』，字數 300 字左右」

### 技巧 2：給範例
**不好**：「我要風格好的文案」
**好**：「請參考這個風格：『老闆，市場調查報告已生成。競爭度中等，但季節性明顯，建議 Q4 前佈局。』」

### 技巧 3：糾正他
**不好**：（沉默，心裡不爽）
**好**：「你剛剛的回覆太長了，我喜歡简短的。請控制在 3 句話以內。」

## 常見用法

### 問問題
```
什麼是阿米巴經營？用白話解釋。
```

### 請他做事
```
幫我整理這份資料，做成表格。
```

### 請他建議
```
我想提高客戶轉換率，有什麼建議？
```

### 請他總結
```
請總結我們昨天討論的重點。
```

## 常見錯誤

### 錯誤 1：一次說太多
不要一次給超過 3 個任務。一次一個，做完再說下一個。

### 錯誤 2：說法模糊
不要說「幫我處理一下」，要說清楚「幫我處理什麼、怎麼處理、何時完成」。

### 錯誤 3：不給他回饋
如果他的回覆不理想，直接說「請重寫，我要更简短的版本」。他會學習你的偏好。

## 壞掉會怎樣
如果你不小心說錯話：
1. 輸入 `/new` 開始新對話（他會忘記上一段對話）
2. 或輸入「忘掉我們剛才說的」重新設定

## 技術補充
- Hermes 的記憶功能保存在 `~/.hermes/memories/`
- 你可以隨時查看他記住了什麼：`hermes memory list`
- 如果你想清除所有記憶：`hermes memory clear`
