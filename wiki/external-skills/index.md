# 外部 Skill 彙整

收錄從網路、社群、GitHub 發現的他人設計的 skill。這些不是知識庫內建的，但值得記錄、參考或引入。

## 使用方式

當遇到對應領域的工作時，可以：
1. 查此頁找到相關 skill
2. 讀取該 skill 的詳細頁面
3. 評估是否引入為 Hermes skill 或直接參考其設計思路

## 新增規則

- 新 skill 進來時，先查此頁是否已有類似項目
- 每個 skill 獨立一篇 wiki，遵循 [[AGENTS.md]] 的 frontmatter 規範
- 標明來源、授權、與內部 skill 的關係
- 定期清理已淘汰或過時的 skill
