# 香料大叔（堉華生技食品）— 私域五階段實戰案例

> 用私域五階段框架，套用在食品品牌的門市導流方案。

## 五階段設計

### 1. 導流

- **做法：** 門市掃碼 → 送香料食譜 PDF

### 2. 分眾

- **做法：** 問「家用/商用/送禮」→ 自動打標籤

### 3. 信任

- **做法：** 每週分享料理教學影片（7:3 原則）

### 4. 轉化

- **做法：** 食譜教學 → 推「新手香料包 990 元」

### 5. 復購 + 擴散

- **復購：** 儲值會員（滿 5000 送 500）
- **擴散：** 分享料理照片到社群 → 標籤「KOC」→ 給專屬折扣

## 商業邏輯分析

### 產品特性

- 低頻消費 → 需要私域提高觸達頻率
- 有教學屬性 → 適合內容經營（料理影片）
- 有 KOC 潛力 → 顧客分享料理照片可轉為口碑

### 變現路徑

```
門市掃碼 → 送食譜 PDF（引流鉤子）
    ↓
問家用/商用/送禮 → 打標籤（分眾）
    ↓
每週料理教學影片（信任內容，7:3 原則）
    ↓
新手香料包 990 元（轉化）
    ↓
儲值會員滿 5000 送 500（復購）
    ↓
分享料理照片 → 標籤 KOC → 專屬折扣（擴散）
```
