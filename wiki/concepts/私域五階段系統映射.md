# 私域五階段 × 我們系統的設計映射

> 核心框架：私域打法的五個階段，直接對應我們系統的模組設計。

## 總覽

| 私域階段 | 系統模組 | 關鍵概念 |
|----------|----------|----------|
| 公域導流 | 引流鉤子模組 | 誘餌可設定，不用改程式 |
| 承接分眾 | 自動標籤引擎 | 標籤自動觸發，不靠人工 |
| 信任內容 | 內容排程引擎 | 內容:促銷 = 7:3 |
| 分層轉化 | 客戶分級邏輯 | 生命週期自動判定 |
| 復購擴散 | 推薦機制 | 雙向獎勵，關係鏈 |

---

## 1. 公域導流 → 引流鉤子模組

**私域打法說：** 要設誘餌導流（掃碼加好友送禮、入群領折抵券）

**我們系統做：**
- Bot 設定 → 歡迎訊息模板
- 掃碼加好友 → 送 X（折價券/免費占卜/小知識）
- 來源追蹤 → 記錄從哪裡來的（IG/FB/門市/推薦）
- 自動拉群 → 依來源分流入不同群組

**關鍵：** 每個群組的「誘餌」可以後台設定，不用改程式。

---

## 2. 承接分眾 → 自動標籤引擎

**私域打法說：** 標籤要自動觸發，不能只靠人工（學中國 SCRM）

**我們系統做：**
客戶進群 → Bot 自動執行：
- 問 1-2 題輕量問卷（選單按鈕，不嚇跑人）
- 記錄關鍵詞（他說了什麼 → 自動打標籤）
- 記錄互動頻率（多久沒回 → 標「待喚醒」）
- 寫入 CRM 資料庫

**標籤維度：**

| 維度 | 標籤範例 | 觸發條件 |
|------|----------|----------|
| 身份/需求 | 命理諮詢 / 產品購買 / 純聊天 | 問卷選擇 |
| 消費力 | 潛在客 / 首購 / VIP | 購買次數+金額 |
| 互動熱度 | 活躍 / 沉默 / 待喚醒 | 最後互動時間 |
| 來源 | IG / FB / 門市 / 推薦 | 歡迎訊息記錄 |

---

## 3. 信任內容 → 內容排程引擎

**私域打法說：** 內容:促銷 = 7:3，固定時段發，不是隨機推播（學完美日記節奏）

**我們系統做：**
後台設定內容排程：
- 每天 09:00 → 早安問候（自動輪替模板）
- 每週三 20:00 → 塔羅抽牌/運勢（互動遊戲）
- 每兩週 → 產品推薦（軟性促銷）
- 每月 → 專屬活動（問卷/抽獎）
- 客戶生日 → 自動祝福（CRM 記錄生日）

**關鍵：** 每個群組的排程可以不同。
- 命理群：塔羅抽牌 + 運勢
- 產品群：新品推薦 + 優惠
- VIP 群：專屬活動 + 一對一關心

---

## 4. 分層轉化 → 客戶分級邏輯

**私域打法說：** 客戶生命週期自動判定，分層鎖客（學星巴克/起士公爵）

**我們系統做：**
CRM 自動判定客戶階段：
- 新客（加好友 0-30 天，未消費）→ 推 introductory 內容
- 首購客（消費 1 次）→ 推儲值優惠
- 常客（消費 3+ 次）→ 升級 VIP，推專屬活動
- VIP（消費 10+ 次或高金額）→ 真人顧問介入
- 沉睡客（30 天無互動）→ 喚醒推播（限時優惠）

**每個群組的轉換邏輯可以不同：**
- 命理群：免費占卜 → 付費諮詢 → 月度訂閱
- 產品群：試用品 → 正價購買 → 儲值會員

---

## 5. 復購擴散 → 推薦機制

**私域打法說：** 推薦關係鏈 + 雙向獎勵（學完美日記/露悠悠）

**我們系統做：**
- 客戶 A 推薦客戶 B → Bot 記錄「A 推薦的」
- B 完成首購 → A 獲得獎勵（折價券/免費占卜）
- 推薦排行榜 → 每月公布 TOP 推薦官
- 高級推薦 → 自動升級 VIP 資格
