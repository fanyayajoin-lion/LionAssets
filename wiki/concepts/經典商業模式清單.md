# 經典商業模式清單（12 種可複用框架）

> **標籤**：#商業模式 #創業框架 #變現模型 #策略設計
> **最後更新**：2026-07-12
> **Lion Score**：長期價值 5 / 顧問可用性 5 / 內容延展性 4 / 可複製性 5 / 變現價值 5

> 這份清單整理了 12 種最經典、可複用性最高的商業模式。每種模式都附上核心邏輯、典型案例和適配新創意的思路。目標：對照自己的創業方向快速匹配，或組合使用多種模式。

---

## 一、訂閱制（Subscription Model）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 用戶按週期付費，持續獲得產品 / 服務使用權 |
| 典型案例 | Netflix、Spotify、SaaS 軟體、會員制健身房 |
| 適用方向 | 內容、軟體、耗材、日用品、專業服務 |
| 新創思路 | 把一次性販售改成「按月 / 按年交付」，提升 LTV（用戶終身價值），現金流更可預測 |

---

## 二、平台 / 雙邊市場（Platform / Two-Sided Market）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 連接供需兩端，自己不持有庫存，靠撮合交易抽成 |
| 典型案例 | 淘寶、Uber、Airbnb、美團 |
| 適用方向 | 本地服務、垂直行業交易、技能撮合、資訊匹配 |
| 新創思路 | 先從一個細分垂直品類切入，解決「雞生蛋」冷啟動問題（先搞定供給端或需求端） |

---

## 三、免費增值（Freemium）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 基礎功能免費，進階功能 / 容量 / 服務付費 |
| 典型案例 | Dropbox、Notion、Canva、大多數 APP |
| 適用方向 | 軟體工具、內容平台、雲服務、SaaS |
| 新創思路 | 免費層用來獲客和建立使用習慣，付費層解決「重度用戶」的痛點 |

---

## 四、剃刀 - 刀片模式（Razor & Blades）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 主設備低價甚至虧本賣，靠後續耗材 / 配件持續賺錢 |
| 典型案例 | 吉列刮鬍刀、印表機 + 墨水、咖啡機 + 膠囊、Kindle + 電子書 |
| 適用方向 | 硬體 + 耗材、設備 + 內容、主機 + 遊戲 |
| 新創思路 | 入口產品降低決策門檻，利潤中心放在高頻複購的消耗品上 |

---

## 五、直銷 / DTC（Direct-to-Consumer）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 跳過經銷商、零售商，品牌直接賣給終端用戶 |
| 典型案例 | Dollar Shave Club、Allbirds、完美日記、Shein |
| 適用方向 | 消費品、美妝、服飾、家居 |
| 新創思路 | 用社群媒體 + 私域流量獲客，掌握用戶數據，毛利更高但獲客成本是關鍵 |

---

## 六、佣金 / 抽成模式（Commission Model）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 每促成一筆交易抽取一定比例或固定金額 |
| 典型案例 | 支付寶、信用卡、外送平台、房仲 |
| 適用方向 | 支付、交易撮合、分銷、經紀業務 |
| 新創思路 | 適合本身不生產商品、只做流量或信任仲介的專案 |

---

## 七、廣告變現（Advertising Model）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 產品免費給用戶用，靠廣告版位賣給商家賺錢 |
| 典型案例 | Google、抖音、微信朋友圈、免費資訊 APP |
| 適用方向 | 內容平台、搜尋、社交、工具類流量產品 |
| 新創思路 | 前提是有足夠大的用戶量和停留時長；小體量專案不建議一開始就押注廣告 |

---

## 八、特許經營 / 加盟（Franchise）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 把成熟的品牌、流程、SOP 授權給加盟商，收取加盟費和分潤 |
| 典型案例 | 麥當勞、星巴克、7-Eleven、連鎖教育 |
| 適用方向 | 餐飲、零售、教育、服務連鎖 |
| 新創思路 | 先跑通單店營利模型和標準化 SOP，再開放加盟擴張 |

---

## 九、按使用付費（Pay-as-you-go / Usage-based）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 用多少付多少，不固定收費 |
| 典型案例 | AWS 雲服務、共享單車、水電瓦斯、API 介接 |
| 適用方向 | 雲服務、基礎設施、共享經濟、API 產品 |
| 新創思路 | 降低用戶試用門檻，適合用量波動大的服務 |

---

## 十、組合銷售 / 綑綁銷售（Bundling）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 把多個產品 / 服務打包成一個套裝整體販售 |
| 典型案例 | Office 全家桶、電信資費套餐、麥當勞套餐 |
| 適用方向 | 軟體、內容、服務組合、零售 |
| 新創思路 | 提高客單價，同時帶動長尾產品的銷量 |

---

## 十一、UGC / 生態分潤（User-Generated Content + Revenue Share）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 平台提供基礎設施，用戶生產內容，收益按比例分潤 |
| 典型案例 | YouTube、抖音創作者分潤、蘋果 App Store、Steam |
| 適用方向 | 內容平台、應用市場、遊戲平台、知識付費 |
| 新創思路 | 關鍵在於建立清晰的分潤機制和創作者扶持體系 |

---

## 十二、長尾模式（Long Tail）

| 維度 | 說明 |
|------|------|
| 核心邏輯 | 不賣爆款，靠大量小眾品類的累計銷量賺錢 |
| 典型案例 | 亞馬遜圖書、Netflix 片庫、淘寶小眾商品 |
| 適用方向 | 電商、數位內容、串流媒體 |
| 新創思路 | 適合數位化、庫存成本極低的品類，滿足細分人群的小眾需求 |

---

## 快速選擇建議

依創業類型匹配優先模式：

| 創業類型 | 優先考慮的模式 |
|----------|----------------|
| 軟體 / 工具類 | Freemium + 訂閱制 |
| 交易撮和 / 仲介 | 平台模式 + 佣金抽成 |
| 硬體 / 設備類 | 剃刀刀片模式 |
| 內容 / 社群類 | 訂閱制 + UGC 分潤 + 廣告 |
| 消費品 / 品牌 | DTC + 訂閱制複購 |
| 線下服務連鎖 | 直營驗證 + 特許經營加盟 |

---

## 關鍵洞察

1. **組合使用是常態**：大多數成功的新創公司不會只用一種模式。例如 SaaS = 訂閱 + Freemium，影音平台 = 訂閱 + 廣告。
2. **模式選擇取決於產品屬性**：硬體天然適合剃刀刀片，軟體天然適合訂閱/Freemium，平台天然適合雙邊市場。
3. **冷啟動是最大挑戰**：平台模式的「雞生蛋」問題、DTC 的獲客成本、UGC 的內容冷啟動，都是落地時的第一道關卡。
