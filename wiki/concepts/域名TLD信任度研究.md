# 域名 TLD 信任度研究 — 台灣傳產老闆的網址觀感

## 核心洞察

對一般人（特別是 40-60 歲傳產老闆）來說，網址的 TLD 差異不在技術層面，而在**信任感知**。

## 信任度排名

| 網址 | 一般人直覺 | 信任度 |
|------|----------|--------|
| `.com` | 正規、大公司、可信 | ⭐⭐⭐⭐⭐ |
| `.com.tw` / `.tw` | 這是台灣公司，比較踏實 | ⭐⭐⭐⭐ |
| `.tech` | 「科技業？工程師？」— 有專業感但偏冷門 | ⭐⭐ |
| `.cloud` | 「雲端服務？AWS 那種？」— 偏 B2B 技術向 | ⭐⭐ |
| 買來的 `xxx.cloud` | 比 subdomain 正式，但還是會有人問「這是什麼網站」 | ⭐⭐⭐ |
| `.hstgr.cloud`（免費 subdomain） | 看到的人不知道是什麼，可能以為是臨時測試站 | ⭐ |

## 關鍵發現

### 台灣市場現實
- 一般人（40-60 歲傳產老闆）**看不懂 TLD 的意義**
- 他們只認 `.com` 和 `.tw`
- `.tech` `.cloud` 對他們來說都是「好像很酷但不太確定在做什麼」
- 反而 `.com.tw` 或 `.tw` 讓他們覺得「這是台灣公司，比較踏實」

### 受眾決定選擇
- **受眾是一般企業主（不是開發者）**：`.tech` 和 `.cloud` 差異不大 → 都不如 `.com.tw` 有說服力
- **受眾是科技圈、年輕一代**：`.tech` 會比 `.cloud` 更貼近「創新」的形象

### 價格差異的真相
- 首年便宜是促銷（`.tech` 常有 $0.99/$1 首年搶市佔率）
- **續訂價才是真相** — .tech 續訂可能回到 $30-50
- 不同 registrar 對同個 TLD 的 markup 可以差很多（Namecheap vs Porkbun vs GoDaddy）

## 對群獅的應用建議

1. 對外展示至少換成自己買的域名，哪怕是便宜的 `.com.tw` 也比 subdomain 看起來正式
2. 受眾是傳產老闆 → 優先 `.com.tw`
3. 受眾是科技圈 → 可考慮 `.tech`
4. 永遠看續訂價，不看首年促銷價
