## 核心概念

業萱的每週靈感題目產出不只是「隨機想 5 個題目」，而是有一套完整的策略框架：

### 1. 素材提取層
從以下來源提取素材：
- `wiki/concepts`：核心概念與方法論
- `wiki/synthesis`：跨領域整合洞察
- `wiki/notes`：日常筆記與觀察
- `hot.md`：熱門話題追蹤
- 每日日記：近期對話中的決策與盲點

### 2. 題目構造層
每個題目必須包含：
- **洞察**：為什麼這個題目重要？打中誰的痛點？
- **類型**：族群共鳴 → 價值沉澱 → 變現代入（三段式結構）
- **切入角度**：從什麼故事或場景開始？
- **時機性**：為什麼「現在」發？

### 3. 評估矩陣
| 維度 | 說明 |
|------|------|
| 痛點強度 | 目標受眾的焦慮程度 |
| 差異化 | 跟競品有什麼不同？ |
| 變現潛力 | 能不能直接導向課程/顧問/提案？ |
| 即時性 | 是不是現在發剛剛好？ |

### 4. 推薦邏輯
- **首選**：痛點最強 + 變現路徑最短
- **次選**：即時性最高 + 能建立真誠形象
