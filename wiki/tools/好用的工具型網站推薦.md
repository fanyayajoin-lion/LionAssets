# 好用的工具型網站推薦

> **標籤**：#工具 #網站推薦 #好站 #資源
> **最後更新**：2026-07-13
> **說明**：收錄被推薦或驗證過好用的工具型網站，供快速查找與回流參考。

---

## 收錄標準

符合以下條件的工具型網站可收錄至此：

1. **功能性明確**：有具體用途（生成、分析、比較、管理等）
2. **免費或可試用**：至少提供免費方案
3. **經過驗證**：老闆實際使用過並認為好用，或值得測試
4. **非重複**：與現有 `wiki/BOSS/AI 工具鏈與基礎設施.md` 不重疊

## 收錄格式

每個工具使用以下格式：

```markdown
### 工具名稱
- **用途**：一行說明
- **網址**：[連結](URL)
- **類型**：免費 / Freemium / 付費
- **狀態**：✅ 已驗證 / 🔶 待測試 / 📝 老闆推薦
- **備註**：使用心得或注意事項
```
