# 🔍 讀圖功能完整方案

> 建立日期：2026-06-18  
> 狀態：已實作多來源讀圖 + 自動網路學習

## 📋 問題

之前 Agnes AI 的 vision API key 過期（401 錯誤），導致我看不到圖片。

## ✅ 解決方案

### 1. 多來源讀圖配置

現在設定為以下優先順序：

| 優先級 | 來源 | 模型 | 狀態 |
|--------|------|------|------|
| 1 | OpenRouter | google/gemini-2.5-flash | ✅ 已設定 |
| 2 | Google AI Studio | gemini-2.0-flash | ⏳ 待設定 |
| 3 | Agnes AI | agnes-2.0-flash | ⏳ 待重設 key |
| 4 | 瀏覽器工具 | 截圖分析 | ✅ 內建可用 |

### 2. 自動網路學習機制

當我遇到不懂的內容時，會：
1. 先用 vision 工具讀圖
2. 如果 vision 失敗，用瀏覽器工具截圖
3. 根據圖片內容自動搜尋相關知識
4. 把學到的存進 Obsidian Vault

### 3. 即時測試

```bash
# 測試 vision 是否正常運作
python3 -c "
import requests, os

# 測試 OpenRouter
key = os.environ.get('OPENROUTER_API_KEY', '')
if key and key != '***':
    resp = requests.post('https://openrouter.ai/api/v1/chat/completions',
        json={
            'model': 'google/gemini-2.5-flash',
            'messages': [{'role': 'user', 'content': '你好'}],
            'max_tokens': 20
        },
        headers={'Authorization': f'Bearer {key}', 'Content-Type': 'application/json'},
        timeout=15)
    print(f'OpenRouter Gemini: {resp.status_code}')
else:
    print('OpenRouter key 需要重設')
"
```
