# 創業檢查清單實戰 — 香料大叔 vs 金飾風險對比

## 核心洞察

用「一人公司創業檢查清單」實際評估兩個案子，得到 45/100 與 20/100 的巨大落差。這證明了檢查清單的最大價值不是「打分」，而是「暴露盲點」。

## 香料大叔（堉華生技食品）：45/100

### 通過項（✅）
市場驗證、商業模式、產品開發、財務規劃、法律合規、行銷策略 — 基礎面尚可。

### 部分通過（⚠️）
市場規模待確認、可複製性待加強、MVP 狀態不明、交付物定義模糊、現金流預測未知。

### 未通過（❌）
- **合作夥伴生態 = 0** — 關鍵合作對象不明、互利模式不清楚
- **持續成長 = 0** — 沒有可重複模組、沒有知識庫

### 高風險
1. 合作夥伴生態完全空白（沒有產業資源無法擴張）
2. 持續成長機制缺失（做一個案子就斷一個，無法累積）

### 建議行動
1. 定義第一個 MVP 交付物（2 週內能做出來展示）
2. 找 1-2 個合作對象小規模試
3. 準備標準顧問合約模板
4. 選一個推廣管道集中打
5. 強制留下 SOP + 案例 + 工具包

## 金飾業：20/100

### 幾乎全數未通過

| 項目 | 香料大叔 | 金飾 |
|------|---------|------|
| 產業了解 | 有基礎 | 完全空白 |
| 痛點確認 | 有假設 | 未驗證 |
| 人脈資源 | 有接觸 | 未知 |
| MVP 方向 | 可定義 | 完全空白 |
| 合作夥伴 | 空白 | 空白 |
| 行銷策略 | 模糊 | 空白 |

### 高風險
1. 完全不了解金飾產業（痛點、預算、法規、人脈）
2. 沒有任何合作夥伴資源
3. 沒有 MVP，不知道能不能做出東西
4. 金飾業高度依賴人脈和信任，進去門檻極高

### 黑獅建議：**先放下金飾**

正確路徑：
```
香料大叔（食品業）→ 累積案例 → 建立 SOP → 拓展到金飾
```

而不是：
```
金飾（完全陌生）→ 盲目探索 → 高風險失敗
```

## 四大洞察

1. **檢查清單的價值在於暴露盲點** — 客戶自認為有方向，但清單顯示執行細節嚴重不足
2. **跨產業擴張的風險被嚴重低估** — 不了解產業就不要碰
3. **「合作夥伴生態」是傳產轉型最容易被忽略的一環** — 傳統產業靠人脈，不是靠廣告
4. **持續成長機制是多人公司容易忽略的** — 一人公司更要注意，不做知識累積就會一直從零開始

## 顧問應用

- 這個對比可以用在提案時展示「為什麼我們要先做食品業，不要同時做金飾」
- 檢查清單本身可以變成一個診斷服務產品
- 45 vs 20 的分數差距是一個很好的自媒體素材：「為什麼你的新案子可能只值 20 分」
