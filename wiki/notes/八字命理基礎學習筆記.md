# 八字命理基礎學習筆記

## 核心概念

### 四柱八字
- 年柱、月柱、日柱、時柱
- 每柱天干+地支 = 8個字
- 日干 = 命主（自己）

### 天干（10個）
甲(陽木)、乙(陰木)、丙(陽火)、丁(陰火)、戊(陽土)、己(陰土)、庚(陽金)、辛(陰金)、壬(陽水)、癸(陰水)

### 地支（12個）
子(水/鼠)、丑(土/牛)、寅(木/虎)、卯(木/兔)、辰(土/龍)、巳(火/蛇)、午(火/馬)、未(土/羊)、申(金/猴)、酉(金/雞)、戌(土/狗)、亥(水/豬)

### 五行生剋
- 相生：金生水 → 水生木 → 木生火 → 火生土 → 土生金
- 相剋：金剋木 → 木剋土 → 土剋水 → 水剋火 → 火剋金

### 十神（以日干為中心）

| 關係 | 同性 | 異性 |
|------|------|------|
| 生我 | 偏印(梟) | 正印 |
| 我生 | 傷官 | 食神 |
| 剋我 | 七殺 | 正官 |
| 我剋 | 偏財 | 正財 |
| 同我 | 劫財 | 比肩 |

### 大運 vs 流年 vs 流月 vs 流日
- **大運**：每十年一轉，人生大環境
- **流年**：每年一轉，當年氣場
- **流月**：每月一轉，月度運勢
- **流日**：每日一轉，當日運勢

### 用神
- 命盤中最需要的五行，用來平衡全局
- 身強 → 喜剋、洩、耗
- 身弱 → 喜生、扶
- 調候：寒冬需火、炎夏需水
