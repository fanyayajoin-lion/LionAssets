# 🐾 群獅知識庫

> 業老闆的數位大腦 — AI 萃取疊加 · 每日提問內化 · 多平台同步
>
> 建立日期：2026-06-17
> 版本：1.0

---

## 目錄結構

```
Lion-Vault/
├── 00_Inbox/          ← 碎片收集箱（原始輸入）
├── 10_Wiki/           ← 知識庫（AI 分類後）
│   ├── concepts/      ← 概念頁（商業模式、方法论）
│   ├── playbooks/     ← 實戰經驗（SOP、操作流程）
│   └── insights/      ← 商業洞察（AI 萃取的見解）
├── 20_Questions/      ← 每日提問（AI 每天問你一個問題）
├── 30_Drafts/         ← 草稿區（AI 產出的文章草稿）
├── 40_Published/      ← 已發布內容（各平台文章）
└── .attachments/      ← 附件（圖片、影片等）
```

## 使用規則

### 碎片輸入
- 格式：`[[日期]]-碎片-[[標籤1]][[標籤2]]`
- 放在 `00_Inbox/`
- AI 會定期萃取分類到 `10_Wiki/`

### 每日提問
- AI 每天晚上 10 點從知識庫選一個主題提問
- 回答寫在 `20_Questions/[[日期]].md`
- AI 會分析你的回答，提取洞察

### 文章產出
- 草稿放在 `30_Drafts/`
- 發布後移到 `40_Published/`
- 包含你的觀點 + 原始素材 + 歷史洞察
