# 每日蒸餾 Workflow

## 目的

把當天新增的 raw、notes、outputs 整理成可查詢、可輸出、可沉澱的知識。

## 步驟

1. 先讀 `index.md`、`hot.md`。
2. 檢查當天新增資料。
3. 將高價值內容整理到 `wiki/notes/` 或既有 wiki 條目。
4. 對重要內容補 Lion Score。
5. 找出可輸出題目與可升級資產。
6. 更新 `hot.md` 的近期熱門、待整理、待輸出、待回流。
7. append `log.md`。

## 產出

- 每日至少一則重點摘要。
- 必要時新增 `wiki/synthesis/`。
- 若同一主題重複三次，提名 Skill、Asset 或 Memory。
