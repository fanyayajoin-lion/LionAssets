# 選題價值判斷 Skill

## 用途

判斷一個靈感、新聞、案例是否值得整理成內容、顧問工具或商業資產。

## 判斷問題

- 能不能幫老闆增加營收？
- 能不能降低老闆對人的依賴？
- 能不能變成顧問方法論？
- 能不能變成課程單元？
- 能不能延伸成文章、短影音、簡報？
- 是否符合傳產、電商、一人公司其中一類客群？

## 輸出

```yaml
topic_value:
  建議: 保留 / 延伸 / 暫存 / 放棄
  理由:
  可輸出形式:
  lion_score:
    變現價值:
    可複製性:
    長期價值:
    顧問可用性:
    內容延展性:
```
