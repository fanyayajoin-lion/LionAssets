# LionAssets — 群獅長期資產庫

與 LionBrain 的分工：
- LionBrain = 短期記憶（日誌、inbox、當前案子）
- LionAssets = 長期資產（Skills、Prompts、SOP、模板）

## 資料夾說明
| 資料夾 | 用途 |
|--------|------|
| skills/internal/ | 小獅自用 Skills |
| skills/client/ | 客戶交付 Skills |
| prompts/gpts/ | GPTs System Prompt 備份 |
| prompts/claude/ | Claude Prompt 模板 |
| prompts/agents/ | Agent 角色定義 |
| sop/ | 可複製 SOP |
| templates/ | 交付模板 |
| archive/ | 舊版本 |
