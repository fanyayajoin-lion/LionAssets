# CRM 模組 — 加購開通 SOP

適合：服務業、顧問、命理師、任何有回頭客的業務

---

## 開通步驟（5 分鐘）

1. 在客戶的 GitHub repo 新增檔案：`clients/README.md`
   內容：`# 客戶資料庫`
2. 重新部署 Zeabur（或等下次自動部署）
3. 完成，CRM 模組自動啟用

---

## 指令說明

| 指令 | 說明 |
|---|---|
| `/newclient 小美` | 建立新客戶檔案 |
| `/client 小美` | 查看客戶完整資料 |
| `/clients` | 列出所有客戶 + 狀態 |
| `@小美 諮詢記錄內容` | 快速追加記錄到客戶檔案 |

---

## 早報自動顯示

開通 CRM 後，每日早報會多一欄：
```
👥 今日待跟進客戶
• 小美｜已諮詢-待推科儀
• 阿華｜新客戶
```

---

## 客戶狀態建議值（命理/諮詢業）

| 狀態 | 說明 |
|---|---|
| 新客戶 | 剛建立，尚未諮詢 |
| 諮詢中 | 正在進行文字諮詢 |
| 已諮詢-待推科儀 | 諮詢完，可以推薦法會 |
| 已成交科儀 | 已報名法會 |
| 長期陪伴 | 固定回頭客 |
| 暫停 | 暫時沒有互動 |

---

## 跟進日期設定

建立客戶後，手動編輯 `clients/小美.md`，在 frontmatter 填入：
```
follow_up: 2026-04-22
```
到期當天早報會自動提醒。
