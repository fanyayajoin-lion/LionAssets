# 群獅管家 Bot — 產品架構

最後更新：2026-04-19

---

## 產品定位

> 從第一次成交，到第 N 次成交，AI 全程陪跑。

```
AI管家（基礎包）
  ↓ 加購
AI業務包 — 找到客戶、追蹤潛客、記錄成交
  ↓ 加購
AI CRM包 — 管理現有客戶、提醒跟進、推動回購
```

---

## 三個模組定義

### AI管家（預裝，所有客戶都有）
- 每天丟東西 → 自動整理分類
- 每天 8:00 早報（客製主題情報 + 案子 + 代辦）
- 每週五週報（Gemini 收集 + Claude 策略分析）
- 問句直接查 Vault 回答
- Zettelkasten 知識卡片自動連結

### AI業務包（加購，`leads/` 資料夾開啟）
- 潛在客戶建檔追蹤
- 成交記錄
- 業務管線狀態（潛客 → 跟進 → 成交 → 轉 CRM）
- 早報顯示「今日待跟進潛客」

### AI CRM包（加購，`clients/` 資料夾開啟）
- 現有客戶完整檔案
- 諮詢/服務記錄
- 跟進日期提醒
- 回購/加購追蹤
- 早報顯示「今日待跟進客戶」

---

## 完整業務流程

```
陌生人
  ↓ AI業務包
潛客建立（leads/）→ 跟進 → 成交
  ↓ AI CRM包
客戶建立（clients/）→ 服務記錄 → 跟進提醒 → 再次成交
  ↓ AI管家
每天整理洞見 → 週報分析 → 優化業務策略
```

---

## 技術架構

```
lion-bot-template（公開模板，Fork 即用）
    ↓ Fork
客戶 repo（填 3 個 config + 7 個環境變數）
    ↓ Zeabur 部署（你的帳號管控）
客戶的 Telegram Bot（24/7 運行）
```

**停用機制：** Zeabur Dashboard → Suspend → Bot 立即停止，資料不刪
