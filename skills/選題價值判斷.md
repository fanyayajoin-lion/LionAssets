# 選題價值判斷 Skill

## 使用時機

判斷一個靈感、新聞、案例、社群題目是否值得整理、輸出或產品化。

## 輸入

- 題目或來源連結
- 目標受眾：傳產老闆、電商創業者、一人公司、顧問客戶
- 預計輸出形式：文章、短影音、簡報、顧問工具、產品功能

## 判斷框架

1. 這件事跟營業額、效率、組織、成交、品牌信任哪一項有關？
2. 老闆看完會不會覺得「這跟我有關」？
3. 是否能接到業萱的人設：有實業背景的商業 AI 摸索者？
4. 是否能延伸成案例、框架、SOP、提案或工具？
5. 是否只是工具新鮮感？如果工具消失，觀點還成立嗎？

## 輸出

- 建議等級：丟棄 / 先收藏 / 值得整理 / 值得輸出 / 值得產品化
- 理由 3 點
- 建議放置位置
- Lion Score
